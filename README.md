# ProgConfig

⚙️ Settings for various dev apps. Currently prefer [Visual Studio Code](https://github.com/microsoft/vscode) on Windows 10.

## Windows AutoHotKey Scripts
[./AHK](./AHK)
- Manipulate Windows using the Mouse
- Open Program shortcuts
- Reading shortcuts
